# Multithreaded Packet Processing Simulator (C++)

This project simulates a packet processing pipeline using a producer–consumer architecture with multiple worker threads.

## Features
- Multithreaded packet processing
- Thread synchronization using mutex and condition variables
- Producer–consumer architecture
- Performance statistics and throughput calculation

## Technologies
- C++
- STL Threads
- Mutex
- Condition Variables

## Build and Run

Compile the program using:
g++ main.cpp -o simulator -pthread
Run the program:
./simulator
